Safe leads and lead changes in competitive team sports

This page is a companion for the article on the statistics of leads in competitive team sports, written by Aaron Clauset, Marina Kogan, and Sid Redner. This page hosts additional information related to the article, including information about the data used, and simulation code for the homogeneous and inhomogeneous Poisson processes.

Journal Reference
A. Clauset, M. Kogan, and S. Redner., "Safe leads and lead changes in competitive team sports." Physical Review E 91, 062815 (2015).


Sports data
The data used in our study were licensed from STATS LLC, which included historical data on every point-scoring event in every league game for NFL, NHL, NBA, and NCAA college football (CFB) over approximately a 10 year period. The terms of the license prevent us from releasing the data publicly. However, as we mention in the article, as of this writing, there are other, publicly available sources of similar data.

The NBA provides detailed play-by-play data on each of its games, going back at least to 1997. For instance, here is a link to the final Cleveland Cavaliers vs. Golden State Warriors game.

ESPN provides similar play-by-play data for NFL games, going back to 2006. For instance, here is a link to the Saints vs. Falcons game on 7 September 2014.

The NHL provides recaps of its games that include the timing and score information (along with fouls, and some other events) for its games. For instance, here is a link for the Blackhawks vs. Lightning game on 13 June 2015.

EPSN provides data on CFB games that is comparable to its NFL data, going back to 2003. For instance, here is a link to the Colorado. vs. Colorado State game on 29 August 2014.

Simulation code
This code runs either the homogeneous or inhomogeneous Poisson process model of scoring dynamics in a competitive team sport. The code requires the specification of the number of seconds and the scoring rate (or function, in the case of the inhomogeneous model), and produces as output the three functions described in the paper.
